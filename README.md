# 📱 Application Ionic Standalone (Calculatrice)

Une application mobile moderne et performante développée avec **Ionic (Angular Standalone Components)**, mettant l'accent sur une interface utilisateur premium, minimaliste et accessible.

---

## ✨ Features

- **Calculatrice Interactive** : Opérations de base (Addition, Soustraction, Multiplication, Division).
- **Design Premium Dark Mode** : Interface minimaliste, neutre et sophistiquée avec un contraste élevé.
- **Accessibilité Optimisée** : Larges zones de clic (tap targets) pour une utilisation mobile fluide.
- **Réactif (Responsive)** : Adapté à toutes les tailles d'écran, des petits smartphones aux grandes tablettes.
- **Composants Standalone** : Architecture Angular moderne et légère.

---

## 🛠️ Tech Stack

- **Framework** : [Ionic Framework](https://ionicframework.com/)
- **Core** : [Angular](https://angular.io/) (Standalone Components)
- **Styling** : SCSS (Sass) avec variables CSS Ionic personnalisées.
- **Platform** : Web / Android / iOS via Capacitor.

---

## 🚀 Getting Started

### Prérequis
- [Node.js](https://nodejs.org/)
- [Ionic CLI](https://ionicframework.com/docs/intro/cli)

### Installation
1. Clonez le dépôt :
   ```bash
   git clone https://github.com/Wissal-badri/Application_Ionic_Standalone.git
   ```
2. Accédez au dossier du projet :
   ```bash
   cd Application_Ionic_Standalone
   ```
3. Installez les dépendances :
   ```bash
   npm install
   ```
4. Lancez l'application localement :
   ```bash
   ionic serve
   ```
